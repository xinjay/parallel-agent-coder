# Agentic SLG Workflow Skills

这是一个面向游戏开发（特别是 SLG 类型）的**多智能体（Multi-Agent）自动化开发工作流**。本仓库包含了一系列为 AI 编程助手设计的核心技能（Skills），通过拆解与并发执行，将人类提供的策划案转化为真实可运行的业务代码。

## 🌟 核心特性 (Core Features)

- **“脑手分离”架构**：高级模型负责统筹与设计（规划师），低级/快速模型负责具体的单点代码实现（大头兵）。
- **DAG 前置依赖推断**：在解析策划案阶段即建立明确的依赖有向无环图，彻底解决并发开发时的代码和接口引用冲突。
- **三级算力动态路由 (3-Tier Model Routing)**：系统根据任务类型自动匹配物理模型池，实现极致的性价比。
  - 🚀 **Ultra-tier (顶级)**：分配给架构搭建与复杂联调。
  - ⚖️ **Mid-tier (中坚)**：分配给核心业务控制器。
  - ⚡ **Fast-tier (极速)**：分配给 UI 视图、CSV/JSON 配置表修改。
- **双引擎兼容**：无缝支持 **Antigravity (agy)** 内置 Subagent API 及 **Claude Code** 终端脚本后台调度，兼具原生一体化与极致透明度。
- **物理沙盒隔离**：利用 Git Worktree 机制为每一次并发任务创建独立工作区，杜绝文件踩踏，合并后再经自动 CI 脚本校验。

## 📦 核心 Skills 概览

### 1. `/task-decompose` (任务拆解引擎)
**职责**：读取策划案，通过对话澄清系统边界，生成自带前置依赖拓扑的 Markdown 任务拆解表以及可独立交付的详细 Spec 文件。

### 2. `/parallel-tasks` (并发执行引擎)
**职责**：读取任务拆解表，进行 Model Discovery 摸底，按 Wave（波次）并发派发不同智力级别的子 Agent 进入工作区写代码。包含三级验收体系（L1 Agent 自查 -> L2 Wave 接口衔接检查 -> L3 冒烟测试）。

## 🚀 快速上手 (Quick Start)

1. 在聊天窗口输入 `/task-decompose {系统名}`，让主控 Agent 读取你放在目录下的策划案并生成拆解表。
2. 确认拆解表和依赖逻辑无误后，输入 `/parallel-tasks {系统名}`。
3. 主控会自动开始建分支、拉后台进程、写代码、合并和编译验证。喝杯咖啡，回来验收成果。

## 🛠 约束与规范
所有由本工作流生成的代码，均严格遵循项目定义的 `AGENTS.md` 编码规范（不使用反射热路径、强制使用 PooledList、统一异步框架等）。
